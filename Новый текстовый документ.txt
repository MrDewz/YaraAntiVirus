Программа AVFramework представляет собой антивирусное приложение, разработанное на платформе .NET WPF (Windows Presentation Foundation). Оно включает в себя функционал для сканирования файлов на наличие вирусов с использованием YARA-правил, интеграцию с базой данных для хранения информации о вирусах, компьютерах и событиях, а также возможность отправки уведомлений по электронной почте. Программа также предоставляет пользовательский интерфейс для отображения обнаруженных вирусов и генерации отчетов.

Основные компоненты программы:
Сканирование на вирусы:

Используется библиотека YaraSharp для применения YARA-правил. YARA — это инструмент для идентификации и классификации вредоносного ПО на основе шаблонов.

Класс YaraScanner.cs отвечает за сканирование файлов и обнаружение вирусов.

База данных:

Программа использует Entity Framework для работы с базой данных. Модель данных (AVModel.edmx) включает такие сущности, как:

viruses: Информация о вирусах.

computers: Данные о компьютерах, на которых проводилось сканирование.

events: События, связанные с обнаружением вирусов.

actions: Действия, предпринятые после обнаружения вирусов.

База данных позволяет хранить историю сканирований, информацию о вирусах и компьютерах.

Пользовательский интерфейс:

WPF используется для создания графического интерфейса. Основные окна:

MainWindow.xaml: Главное окно программы.

DetectedViruses.xaml: Окно для отображения обнаруженных вирусов.

ReportVirusWindow.xaml: Окно для создания отчетов о вирусах.

Для визуализации данных используется библиотека LiveCharts.Wpf, которая позволяет строить графики и диаграммы.

Уведомления и логирование:

Программа поддерживает отправку уведомлений по электронной почте с использованием библиотек MailKit и MimeKit.

Логирование событий (например, обнаружение вирусов) реализовано в классе Logging.cs.

Интеграция с системным треем:

Используется библиотека Hardcodet.NotifyIcon.Wpf для работы с иконкой в системном трее. Это позволяет программе работать в фоновом режиме и отображать уведомления.

Дополнительные функции:

Класс DiskSearcher.cs отвечает за поиск файлов на дисках.

Класс InfoComputer.cs собирает информацию о компьютере.

Класс AutoRunClass.cs, вероятно, отвечает за автозапуск программы.
